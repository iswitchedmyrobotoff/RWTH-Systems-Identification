void CancelEvent::handleEvent ( )
overridevirtual

Führt einen Verzögerungsvorgang aus.

Diese Methode blockiert für eine definierte Dauer (2500 ms) durch eine Schleife mit kurzen Verzögerungen. Sie dient dazu, eine Unterbrechung oder Wartezeit zwischen Aktionen zu simulieren.

Note
Die Methode verwendet HAL_Delay() für die Verzögerung.
