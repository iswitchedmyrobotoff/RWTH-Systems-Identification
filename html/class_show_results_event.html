void ShowResultsEvent::handleEvent ( )
overridevirtual

Zeigt Ergebnisse an und wartet auf Benutzereingabe zum Fortfahren.

Setzt den aktuellen Zustand auf SHOW_RESULTS und blockiert für maximal 2500 ms, während auf eine Benutzereingabe gewartet wird. Die Schleife wird abgebrochen, sobald die Variable buttonPressed auf 1 gesetzt wird.

Note
Die Methode dient dazu, eine Pause einzulegen, bis der Benutzer interagiert.
